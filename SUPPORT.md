### Common Issues

#### ZIP file won't extract

- Ensure the file is a valid `.zip` archive (not `.rar`, `.7z`, etc.)
- Check that the file isn't corrupted
- Try a smaller file to verify the app works
- Clear browser cache and retry

#### Upload appears stuck

- Check your browser's developer console for errors
- Ensure sufficient available memory for large files
- Try a different browser

#### Downloaded files are corrupted

- Ensure you're downloading the correct file
- Try downloading individual files instead of a batch ZIP
- Check your available disk space

### Reporting Issues

When reporting a bug, please include:

1. Browser name and version
2. Operating system
3. Steps to reproduce
4. Expected vs actual behavior
5. Console errors (if any)
6. File size (approximate)
